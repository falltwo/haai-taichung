# 醫療繁榮的假象 — 台中市醫療可及性 HAAI 分析

114-2 資料視覺化期末專案 · Medical Desert vs. Aging Crisis

把台中市 29 個行政區的「醫療資源供給」攤開對上「老化需求」與「就醫距離」,用 **HAAI 醫療可及性缺口指數** 揭穿「醫療繁榮」表象下的結構性不平等:醫療量能全台前三,但最需要醫療的人,反而最難就醫。

## 互動說明

點擊「各區 HAAI V3 排名」表中的任一行政區,即時輸出該區的**現況診斷 / 介入建議(謀) / 驗證指標(斷)**。

## 核心發現

- **北屯區**:16.9 萬急診需求 vs 159 床(每床 1,067 人次),任何承載假設下都遠遠不足 = 結構性資源錯配。
- **和平區**:距最近醫學中心 39.4 公里,屬距離型可及性危機,與市區的「超載」問題本質不同。
- 全市 10 區無在地一般急診床,居民須跨區就醫。

## 資料來源

台中市 Open Data 醫院基本資料、衛福部醫院一般病床數、台中市民政局人口結構、衛福部醫療機構人員統計、衛福部縣市別急診就醫概況、g0v/twgeojson 行政區界線。

## 方法論限制(誠實揭露)

- 分區急診需求係以費米加權(高齡權重 ×3)由全市值推估分配,非各區實測。
- 急診資料年(2024)與人口資料年(2026)不一致。
- 就醫距離採行政區中心點直線距離代理,非行車距離。
- 清水、石岡標為「跨區依賴」係因院所為精神/慢性療養醫院,一般急性病床為 0,非全無醫療設施。
- 跨區就醫流向為結構估算,非健保實際就醫移動紀錄。
